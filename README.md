# Team Leave Calendar

A full-stack web application for managing team leaves and vacations, built with **Spring Boot** (Backend) and **Angular** (Frontend).

## Project Structure

This repository is containing both the backend and frontend services:

* **/team-leave-calendar** - The core project directory containing the application logic.

## Tech Stack

* **Backend:** Java, Spring Boot, Spring Data JPA, REST API
* **Frontend:** Angular, TypeScript
